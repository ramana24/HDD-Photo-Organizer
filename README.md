# Photo Organizer Pro

Organize 10 years of photos by date (Year/Month/Day), with options to copy or move files and manage duplicates.

## Features
- Year/Month/Day folder structure based on EXIF date or file time
- Copy or move originals
- Duplicate checks: skip, rename, overwrite
- Progress logs and preview
- Windows installable via PyInstaller or auto-py-to-exe
